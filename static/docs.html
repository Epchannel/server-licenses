📌 Tổng quan

License Server cung cấp 2 API chính để tool kiểm tra bản quyền:

  • Verify — Xác minh license key khi tool khởi động
  • Heartbeat — Gửi tín hiệu định kỳ khi tool đang chạy (5-10 phút/lần)

Base URL

https://your-server-domain.com
⚠️ Thay your-server-domain.com bằng domain thực tế khi deploy. Khi dev local: http://localhost:8000

🚀 Quick Start — Tích hợp nhanh

Chỉ cần 3 bước để tích hợp kiểm tra bản quyền vào tool:

Bước 1: Xác định TOOL_CODE

Mỗi tool có một TOOL_CODE duy nhất. Ví dụ:

TOOL_CODE = "VER_PHONE_VEO"     # Ver Phone Veo 3
TOOL_CODE = "CHECK_VEO3"        # Check Ultra Veo 3
TOOL_CODE = "PICK_VEO3"         # Pick User Veo 3
TOOL_CODE = "CHANGE_PASS_VEO3"  # Change Pass Veo 3
💡 Khi tạo tool mới trên Admin Panel, code sẽ tự động tạo endpoint: /api/v2/{TOOL_CODE}/verify

Bước 2: Gọi API Verify khi khởi động

import requests, uuid

LICENSE_SERVER = "http://localhost:8000"
TOOL_CODE = "VER_PHONE_VEO"
DEVICE_ID = str(uuid.getnode())  # Lấy MAC address làm device ID

def verify_license(license_key):
    response = requests.post(
        f"{LICENSE_SERVER}/api/v2/{TOOL_CODE}/verify",
        json={
            "license_key": license_key,
            "device_id": DEVICE_ID,
            "device_name": "PC-User"
        }
    )
    data = response.json()
    return data["valid"], data["message"]

🔑 API 1: Verify License

POST /api/v2/{tool_code}/verify

Xác minh license key. Gọi khi tool khởi động. Tự động bind device lần đầu.

Request Body (JSON)

FieldTypeBắt buộcMô tả
license_keystring✅ Bắt buộcLicense key cần kiểm tra
device_idstringKhông bắt buộcID thiết bị (MAC, UUID...). Nếu gửi sẽ tự bind lần đầu.
device_namestringKhông bắt buộcTên thiết bị (hiển thị trên admin)

Content-Type: application/json

    "device_name": "Laptop-HiepPham"

💓 API 2: Heartbeat

POST /license/heartbeat

Gửi tín hiệu định kỳ (5-10 phút) khi tool đang chạy. Server dùng để theo dõi usage và kiểm tra license còn hiệu lực.

Request Body (JSON)

FieldTypeBắt buộcMô tả
license_keystring✅ Bắt buộcLicense key
device_idstring✅ Bắt buộcID thiết bị (phải khớp với lần verify)

Response

// Thành công
{ "valid": true, "message": "Heartbeat OK", "timestamp": "2026-05-08T01:00:00", "signature": "..." }

// Thất bại
{ "valid": false, "message": "Bản quyền đã hết hạn", "timestamp": "...", "signature": "..." }
{ "valid": false, "message": "Thiết bị không khớp", "timestamp": "...", "signature": "..." }

📋 Response Format Chung

FieldTypeMô tả
validbooleantrue = license hợp lệ, false = không hợp lệ
messagestringThông báo kết quả (tiếng Việt)
expire_atstring (ISO)Ngày hết hạn (chỉ khi valid=true hoặc hết hạn)
featuresarrayDanh sách features được cấp
user_namestringTên user sở hữu license
signaturestringChữ ký HMAC để chống giả mạo response
💡 Quy tắc đơn giản: Chỉ cần kiểm tra data["valid"] == True là đủ.

📱 Device Binding

Khi gửi device_id trong request verify:

  • Lần đầu: Server tự động gán device_id vào license (bind)
  • Lần sau: Nếu device_id khác → trả về valid: false
  • Reset: Admin có thể reset device binding trên panel

Cách tạo Device ID

# Python — dùng MAC address
import uuid
DEVICE_ID = str(uuid.getnode())

# Python — dùng kết hợp MAC + hostname
import socket
DEVICE_ID = f"{uuid.getnode()}-{socket.gethostname()}"

# C# — dùng MAC address
using System.Net.NetworkInformation;
string deviceId = NetworkInterface.GetAllNetworkInterfaces()
    .Where(n => n.OperationalStatus == OperationalStatus.Up)
    .Select(n => n.GetPhysicalAddress().ToString())
    .FirstOrDefault() ?? Environment.MachineName;

🐍 Code mẫu đầy đủ — Python

Copy class này vào project tool, chỉ cần thay LICENSE_SERVER và TOOL_CODE:

"""
License Manager — Tích hợp EP Channel License Server
Copy file này vào project tool, dùng class LicenseManager.
"""
import requests
import uuid
import socket
import threading
import time
import sys


class LicenseManager:
    def __init__(self, server_url, tool_code):
        """
        Args:
            server_url: URL của license server. VD: "http://localhost:8000"
            tool_code:  Code của tool trên server. VD: "VER_PHONE_VEO"
        """
        self.server_url = server_url.rstrip("/")
        self.tool_code = tool_code
        self.device_id = f"{uuid.getnode()}-{socket.gethostname()}"
        self.device_name = socket.gethostname()
        self.license_key = None
        self.user_name = None
        self.expire_at = None
        self._heartbeat_thread = None

    def verify(self, license_key):
        """Xác minh license key. Trả về (True, data) hoặc (False, message)."""
        self.license_key = license_key
        try:
            resp = requests.post(
                f"{self.server_url}/api/v2/{self.tool_code}/verify",
                json={
                    "license_key": license_key,
                    "device_id": self.device_id,
                    "device_name": self.device_name
                },
                timeout=10
            )
            data = resp.json()

            if data.get("valid"):
                self.user_name = data.get("user_name")
                self.expire_at = data.get("expire_at")
                return True, data
            else:
                return False, data.get("message", "License không hợp lệ")
        except requests.exceptions.ConnectionError:
            return False, "Không thể kết nối tới license server"
        except Exception as e:
            return False, f"Lỗi: {str(e)}"

    def start_heartbeat(self, interval=300):
        """Bắt đầu gửi heartbeat mỗi {interval} giây (mặc định 5 phút)."""
        if not self.license_key:
            return

        def _loop():
            while True:
                try:
                    resp = requests.post(
                        f"{self.server_url}/license/heartbeat",
                        json={
                            "license_key": self.license_key,
                            "device_id": self.device_id
                        },
                        timeout=10
                    )
                    data = resp.json()
                    if not data.get("valid"):
                        print(f"⚠️ Heartbeat failed: {data.get('message')}")
                except:
                    pass
                time.sleep(interval)

        self._heartbeat_thread = threading.Thread(target=_loop, daemon=True)
        self._heartbeat_thread.start()

    def verify_and_start(self, license_key):
        """Verify + tự động start heartbeat. Thoát app nếu thất bại."""
        valid, result = self.verify(license_key)
        if valid:
            print(f"✅ Bản quyền hợp lệ! User: {self.user_name}")
            print(f"   Hết hạn: {self.expire_at}")
            self.start_heartbeat()
            return True
        else:
            print(f"❌ {result}")
            sys.exit(1)


# ============ CÁCH SỬ DỤNG ============
if __name__ == "__main__":

        server_url="http://localhost:8000",

🔷 Code mẫu — C#

using System;
using System.Net.Http;
using System.Text;
using System.Text.Json;
using System.Threading;
using System.Threading.Tasks;
using System.Net.NetworkInformation;
using System.Linq;

public class LicenseManager
{
    private readonly string _serverUrl;
    private readonly string _toolCode;
    private readonly string _deviceId;
    private string _licenseKey;
    private readonly HttpClient _http = new HttpClient();

    public string UserName { get; private set; }
    public string ExpireAt { get; private set; }

    public LicenseManager(string serverUrl, string toolCode)
    {
        _serverUrl = serverUrl.TrimEnd('/');
        _toolCode = toolCode;
        _deviceId = GetDeviceId();
    }

    private string GetDeviceId()
    {
        var mac = NetworkInterface.GetAllNetworkInterfaces()
            .Where(n => n.OperationalStatus == OperationalStatus.Up)
            .Select(n => n.GetPhysicalAddress().ToString())
            .FirstOrDefault();
        return $"{mac ?? "unknown"}-{Environment.MachineName}";
    }

    public async Task<(bool valid, string message)> VerifyAsync(string licenseKey)
    {
        _licenseKey = licenseKey;
        var payload = JsonSerializer.Serialize(new {
            license_key = licenseKey,
            device_id = _deviceId,
            device_name = Environment.MachineName
        });
        var content = new StringContent(payload, Encoding.UTF8, "application/json");
        var resp = await _http.PostAsync($"{_serverUrl}/api/v2/{_toolCode}/verify", content);
        var json = JsonDocument.Parse(await resp.Content.ReadAsStringAsync());
        var root = json.RootElement;

        bool valid = root.GetProperty("valid").GetBoolean();
        string message = root.GetProperty("message").GetString();

        if (valid) {
            UserName = root.TryGetProperty("user_name", out var u) ? u.GetString() : null;
            ExpireAt = root.TryGetProperty("expire_at", out var e) ? e.GetString() : null;
        }
        return (valid, message);
    }

    public void StartHeartbeat(int intervalSeconds = 300)
    {
        var timer = new Timer(async _ => {
            try {
                var payload = JsonSerializer.Serialize(new {
                    license_key = _licenseKey,
                    device_id = _deviceId
                });
                var content = new StringContent(payload, Encoding.UTF8, "application/json");
                await _http.PostAsync($"{_serverUrl}/license/heartbeat", content);
            } catch { }
        }, null, TimeSpan.Zero, TimeSpan.FromSeconds(intervalSeconds));
    }
}

✅ Best Practices

  • Verify 1 lần khi khởi động — Không verify mỗi action, chỉ cần lúc start
  • Heartbeat 5-10 phút/lần — Dùng background thread, daemon=True
  • Xử lý offline — Nếu không kết nối được server, cho phép chạy tạm (grace period) hoặc thoát
  • Device ID ổn định — Dùng MAC address, không dùng random UUID (sẽ thay đổi mỗi lần chạy)
  • Timeout — Set timeout 10s cho requests để không block UI
  • Không hardcode key — Cho user nhập key hoặc lưu trong file config
  • Kiểm tra valid — Chỉ cần check data["valid"] == True, không cần parse message
